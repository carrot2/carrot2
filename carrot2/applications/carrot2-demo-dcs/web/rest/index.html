Carrot Search Document Clustering Server

You can invoke clustering by sending a multipart/form-data POST request to the Document Clustering Server, providing input data in the c2stream form parameter. An optional parameter named algorithm may specify the algorithm to use for clustering (note that this parameter must precede the actual data stream). If nothing is given, the default algorithm is used.

You can use the forms below to trigger clustering from your browser.

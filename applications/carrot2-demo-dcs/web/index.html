Document Clustering Server

Document Clustering Server (DCS) provides the following services:

HTTP POST
Clusters XML input data provided in a HTTP POST request. Format of the data is the following: 
<searchresult>
  <query>new york hotels</query>
  <document>
    <title>NewYork.com</title>
    <snippet>
      guide to New York City with links to hotels, attractions, 
      shopping, restaurants, and more. NewYork.com also features 
      local New York news, arts, and sports.
    </snippet>
  </document>

  <document>
      ...
  </document>
  
  ...
</searchresult>
Please see this page for an example HTML form showing how to post queries and how to select the clustering algorithm dynamically (if there is more than one installed).
